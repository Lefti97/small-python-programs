# sorting-visualization

Sorting Visualization using Pygame.

Instructions:
Execute py sortVis.py

SPACE: Pause, 1: Shuffle, 2: Asc Sort, 3: Desc Sort

Q: Insertion Sort

W: Bubble Sort

E: Quick Sort

R: Selection Sort

T: Merge Sort

Y: OddEven Sort

U: Shell Sort

... more sorting algorithms coming ...
